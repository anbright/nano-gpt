# NanoGPT

## Implementation of NanoGPT from Andre Karpathy's video

See [github](https://github.com/karpathy/nanoGPT) and [YouTube](https://www.youtube.com/watch?v=kCc8FmEb1nY&list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ&index=7)

Implements a character-level language model on the [Tiny Shakespeare dataset](https://huggingface.co/datasets/karpathy/tiny_shakespeare).
